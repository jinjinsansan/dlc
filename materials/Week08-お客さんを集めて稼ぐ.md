# Week 8: お客さんを集めて稼ぐ

> **このWeekのゴール**: SNS で告知して、最初の 1 人のユーザー（or 売上）を獲得する
> **所要時間**: 動画 50 分 + 教材 90 分 + 宿題 30 分
> **このWeekで身につく力**: 「作ったものを世に届ける」集客の基本。これが最後で、最も重要

---

## このWeek を始める前に

### 多くの個人開発者が落ちる「集客の谷」

> **作っただけでは、誰も来ません**

これは AI 個人開発者全員が必ずぶつかる現実です。
URL を作って「公開しました!」と宣言しても、**最初の 1 ヶ月は 1 日 5 アクセスとか普通**。

### 集客は「作る」より時間と工夫が要る

```
作る期間: 8 週間
集客期間: 永遠
```

**事実**: 作るのは速いです。難しいのは **「使ってもらう」** こと。

ただし、Week 8 で **最初の 1 人** を獲得する方法は確立されています。
ここで踏み出せば、2 人目・3 人目は自然と続きます。

## レッスン 8-1: 集客の全体像（10 分）

### 集客チャネル 4 種

| チャネル | 即効性 | 持続性 | 個人向け | コスト |
|---------|--------|--------|---------|-------|
| **SNS（X / Instagram / TikTok）** | ⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 無料 |
| **ブログ・note** | ⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 無料 |
| **広告（Google / Meta）** | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ | 有料 |
| **口コミ・紹介** | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 無料 |

### 個人開発者の鉄板パターン

```
SNS（X 中心）で発信
    ↓
ブログ・note で深掘り記事を書く
    ↓
記事から SNS フォロワー獲得
    ↓
SNS でサービスを告知
    ↓
最初のお客さんが来る
    ↓
お客さんが SNS で紹介してくれる（口コミ）
```

このループを **「最初の 1 人」から始める** のが Week 8 のゴール。

### 「集客 = しんどい営業」ではない

集客と聞くと **「営業電話」「セールス」** を思い浮かべるかもしれません。
個人開発の集客は **全く違います**。

```
やること:
- 自分が作ったものを「これ作りました!」と SNS に投稿
- どう作ったかをブログに書く
- 困っている人を見かけたら「これ使えそう」とコメント

やらないこと:
- 知らない人に DM 営業
- 怪しい広告を打つ
- お願いして「買ってください」と言う
```

**「価値を届ける」** が個人開発の集客の本質です。

## レッスン 8-2: X で告知する方法（15 分）

### X（旧 Twitter）が最強な理由

- 個人開発者・AI に興味ある人が多い
- フォロワー 0 でも見つけられやすい（リプ・引用・トレンド）
- 短文で発信できる（時間がかからない）

### プロフィール設計

#### テンプレ

```
名前: 山田太郎 | AI 個人開発で月10万を目指す

プロフィール:
40 代会社員。AI で人生を変える挑戦中。
8 週間で初めて作ったサービスをリリース 🚀
作ったもの: [サービス URL]
日々の学びと作ったものをシェアします。

【発信テーマ】
- AI 個人開発の試行錯誤
- ノーコード制作日記
- 副業から本業化の道のり
```

#### プロフィール 5 つの鉄則

1. **名前**: ブランド or 本名（覚えやすく）
2. **肩書き**: 「○○を目指す」と未来形で書くと共感されやすい
3. **実績**: 数字で示せると最強（フォロワー数・売上等）
4. **発信内容**: 何を発信するか宣言
5. **URL**: 自分のサービスへ誘導

### 固定ツイート

プロフィール画面の一番上に固定する投稿。
これが **「初対面の自己紹介」** になります。

#### 鉄板テンプレ

```
8 週間前まで、コードを 1 行も書けなかった私が、
AI 個人開発で初めて作ったサービスをリリースしました。

▼ サービス
[URL]

▼ 機能
- ○○
- ○○
- ○○ (AI 機能)

無料プランあります。ぜひ試してみてください。
```

### 「リリース告知ツイート」の書き方

#### NG 例

```
新サービス出ました! [URL]
→ 反応ゼロ
```

#### 良い例

```
🚀 リリースしました!

「○○な人」のための「○○できる」サービスを作りました。

【特に試してほしい機能】
1. AI が○○してくれる
2. 月○回の○○で時間節約
3. 月○○円で使い放題

▼ 試してみる
[URL]

質問・感想あれば返信ください 🙏
```

### コツ: 「数字」「ストーリー」「証拠」を入れる

```
✅ 数字: 「8 週間で」「月 980 円」「すでに 50 人が利用」
✅ ストーリー: 「会社員の私が」「全くコード書けない 40 代」
✅ 証拠: スクリーンショット・動画・データ
```

これらが入っているツイートは、**反応が 5 倍以上** 違います。

## レッスン 8-3: note で集客記事を書く（15 分）

### note が個人開発に向いている理由

- ストック型（記事が長く残る）
- SEO で検索流入が来る
- ファンができやすい
- 有料記事も書ける

### 鉄板記事タイプ「○○を作ってみた」

#### テンプレ

```
タイトル: 【全行程公開】8 週間で○○を作ってリリースしました

導入:
- 自己紹介
- なぜこれを作ったか（個人的なストーリー）

本文:
1. なぜこのサービスを作ったか
2. 実装した機能・使った技術
3. 8 週間の試行錯誤（具体例 3 つ）
4. 完成して気づいたこと
5. 今後の展開

CTA:
- サービス URL
- フォロー誘導
```

#### 良い例タイトル

```
✅ 「AI で何でも作れる時代を体感した 8 週間の記録」
✅ 「コードゼロの 40 代会社員が、月980円のサービスをリリースするまで」
✅ 「外注したら 100 万円のサービスを、Claude Code で 5 万円で作った話」
✅ 「8 週間で個人開発デビューしたら、人生変わった」
✅ 「私が AI 副業で『最初の 1 円』を稼ぐまでにやったこと全部」
```

#### Bad タイトル

```
❌ 「サービスをリリースしました」（弱い）
❌ 「○○の使い方」（CTA 不足）
❌ 「私の自己紹介」（誰も読まない）
```

### 文字数の目安

- **2,000〜3,000 字** が読みやすい
- 見出しを 3〜5 個入れる
- スクリーンショット・図を入れる
- 最後に必ず CTA（サービス URL）

## レッスン 8-4: AI に告知文を書かせる（10 分）

### この技を覚えると、集客が 10 倍楽になる

毎日 SNS 投稿するのは大変です。AI に書いてもらいましょう。

#### Claude にお願い

```
私のサービスの X 告知文を 5 パターン書いて。

【サービス情報】
名前: ○○
ターゲット: ○○な人
解決する悩み: ○○
特徴: ○○
価格: 無料 / 月○○円
URL: ○○

【トーン】
親しみやすく、押し付けがましくない
1 投稿 200 字以内
ハッシュタグ 2〜3 個

【パターン分け】
1. 価値訴求型（「こんなことができます」）
2. ストーリー型（「私が作った理由」）
3. 質問投げかけ型（「○○で悩んだことありませんか?」）
4. 数字訴求型（「すでに○○人が利用」）
5. ビフォーアフター型（「○○だった私が」）
```

→ Claude が 5 パターンの投稿文を生成。

### 自分のサービスの紹介文も AI に

```
私のサービスを 100 字で紹介する文章を書いて。
```

```
私のサービスを 30 秒の動画で紹介するスクリプトを書いて。
```

```
私のサービスのキャッチコピーを 10 個提案して。
```

### note 記事も AI が書く

```
タイトル: 「8 週間で AI 個人開発デビューした記録」
これで note 記事を 2,500 字で書いて。

【含めて欲しい要素】
- 私の経歴: 40 代会社員、コード経験ゼロ
- きっかけ: AI Builders Lab 参加
- 8 週間で何をやったか（時系列で）
- 感動した瞬間
- 完成したサービス: ○○
- 次にやりたいこと
- CTA: フォロー + サービス URL

【トーン】
- 等身大の言葉で
- 飾らない正直な感じで
- 数字や具体例を多めに
```

→ AI が記事を書いてくれる。**自分で 1 文字も書かなくても、立派な記事が出来る**。

## レッスン 8-5: AI 自動投稿の仕組み（15 分）

### 毎日投稿を「自動化」する

```
毎日 1 投稿を続けたい
↓
でも忙しい
↓
AI が自動で投稿を作って、自動で X に投稿してくれたら最高
↓
作れます。Week 8 で実装。
```

### 仕組み

```
1. 朝 7 時にサーバーで Claude API を呼ぶ
   → 今日の投稿文を生成
2. 自動で X API に POST
   → 投稿される
3. 自分は何もしなくて OK
```

### 実装方法

#### 1. X API キー取得

1. [https://developer.x.com](https://developer.x.com) でアカウント
2. プロジェクト作成 → API Keys 取得（無料枠あり）
3. `.env.local` に追加:
   ```
   X_API_KEY=xxx
   X_API_SECRET=xxx
   X_ACCESS_TOKEN=xxx
   X_ACCESS_TOKEN_SECRET=xxx
   ```

#### 2. Claude に頼む

```
毎日朝 7 時に X に自動投稿する仕組みを作って。

【投稿内容】
Claude API でその日の投稿文を生成。
パターンは 4 種類でローテーション:
1. 価値提供（AI 開発のコツ）
2. ストーリー（私の体験談）
3. 質問投げかけ
4. サービス告知

【投稿先】
私の X アカウント
（API キーは .env.local に設定済み）

【スケジュール実行】
Vercel Cron Jobs を使う

【失敗時】
ログを残す
私のメールに通知
```

→ Claude が全工程を実装。

### 効果

- 毎日「今日何投稿しよう...」と悩む時間がゼロに
- 投稿継続が出来る = フォロワーが増える
- 結果としてサービス URL のクリックも増える

私の場合、これで月の投稿数が **3 投稿 → 90 投稿** になり、フォロワーが半年で **10 倍** になりました。

## レッスン 8-6: 最初の 1 人を獲得する方法（10 分）

### 「最初の 1 人」が一番難しい

最初の 1 人を獲得するまでが、実は一番大変です。
2 人目以降は **「誰かが使っているサービス」** という証拠ができるので楽になります。

### 最初の 1 人の獲得法

#### 王道: 友人・知人に頼む

```
「○○ってサービス作ったんだけど、ちょっと試してくれない?
無料プランあるから。
良かったら感想教えて、悪かったら直すから」
```

これが **最も即効性ある** 方法です。
お金は要らない。**「触ってくれた人がいた」という事実** が大事。

#### コミュニティで紹介

AI Builders Lab のコミュニティに完成サービスを投稿:
```
[サービスリリース報告]

8 週間でこれを作りました!
URL: ○○

【特徴】
- ○○
- ○○

仲間に試してもらえると嬉しいです 🙏
```

→ 受講生同士で **「相互利用」「相互フォロー」「相互レビュー」** が発生します。これが最初の 5〜10 人になります。

#### Twitter / X 廃トレンドで便乗

「#つくってみた」「#個人開発」「#リリース報告」 などのハッシュタグを付けて投稿。
コミュニティの目に留まりやすい。

#### 競合サービスの「不満ツイート」を探す

X の検索で「○○ 使いにくい」「○○ 高い」を検索。
そこにリプライで:
```
私が作ったサービスが解決できそうです。
無料プランあります → [URL]
よろしければ試してみてください。
```
→ 押し付けがましくならない範囲で。

#### Product Hunt（中級）

[Product Hunt](https://www.producthunt.com/) で英語圏に告知。
うまく行けば一気に数百ユーザー獲得も。

## レッスン 8-7: これからのロードマップ（10 分）

### 卒業後 6 ヶ月で目指すもの

```
Month 1: 卒業直後
- ユーザー数: 5〜20 人（友人・コミュニティ中心）
- 売上: 数千円〜1 万円
- やること: 改善・SNS 発信継続

Month 2-3:
- ユーザー数: 50〜200 人
- 売上: 月 3〜10 万円
- やること: ブログ記事を 5 本書く・SNS フォロワー獲得

Month 4-6:
- ユーザー数: 200〜1,000 人
- 売上: 月 10〜30 万円
- やること: 第 2 のサービス着手・コミュニティ運営

1 年後:
- 月収 30 万円以上
- 副業から本業化を検討できる状態
- 第 2・第 3 のプロダクトもリリース
```

これは **平均的な受講者の到達ライン** です。あなたなら、もっと早くもっと大きく到達できる可能性があります。

### 第 2 のプロダクトの作り方

第 1 プロダクトの集客が落ち着いたら、**第 2 プロダクト** を作りましょう。

#### 第 2 プロダクトを選ぶコツ

```
- 第 1 プロダクトのユーザーが「ついでに欲しい」もの
- 第 1 プロダクトの「失敗」を活かせる別ジャンル
- 自分が今度こそ本当にやりたかったもの
```

複数プロダクトを持つと、**収益の柱が複数** できて精神的に楽になります。

### 収益の伸ばし方 3 つ

#### ① 単価アップ
```
月 980 円 → 月 1,980 円に値上げ
理由: 機能追加した、価値が上がった
新規ユーザーから順次値上げ。既存は据え置きでも OK
```

#### ② プレミアムプラン追加
```
既存: 月 980 円
追加: 月 2,980 円のプロプラン（高度機能・サポート付き）
```

#### ③ 関連プロダクト追加
```
本体サービス: 月 980 円
関連教材: 9,800 円（一括）
コーチング: 月 30,000 円
```

### 「次の Claude Code 教材」も用意してます

AI Builders Lab を卒業した後、**「もっと深く学びたい」** という方向けに:

- 卒業生限定コミュニティ（月 4,980 円〜）
  - 新しいフレーズ集の追加配布
  - 月 1 回のオンライン講座
  - 受発注ボード（仲間と仕事を回せる）
- 個別コーチング（要相談）
- 第 2 期生向け Tier アップ

詳しくは卒業後にご案内します。

## トラブルシューティング

### Q1. SNS で投稿しても反応がゼロです

**A**: 最初の 1 ヶ月は当たり前です。継続が全て。
- 毎日 1 投稿を 30 日続ける
- 内容は「自分の試行錯誤」「気づき」「失敗談」
- フォロー外の人にも届けるためハッシュタグ 2〜3 個

### Q2. 友達に「すごいね」と言われるが、誰も使ってくれません

**A**: 「すごい」と「使う」は違います。
- 友達は「友情」で「すごい」と言うが、使う動機がない
- 「無料プランで○○ができる、3 分で試せる」と具体的にメリットを伝える
- 「使ってフィードバックくれたらお礼に○○あげる」も有効

### Q3. お金を取るのが恥ずかしいです

**A**: そう感じる人ほど **「無料で使ってもらう」 → 「価値を実感してもらう」 → 「有料プランへアップグレード提案」** の段階を踏む。最初から有料にせず、無料プランから始めるのが王道。

### Q4. 競合がたくさんあって勝てる気がしません

**A**:
- 競合がたくさんある = 「市場がある」証拠（むしろ良いこと）
- 「全員に勝つ」のではなく「ニッチで 1 番」になる
- ターゲットを絞る:「20 代 OL 限定」「副業ワーカー専用」等

### Q5. AI 自動投稿の設定が難しい

**A**: 後回しで OK。最初は手動投稿で十分。
慣れてきたら Claude に「X 自動投稿の Cron 設定して」と頼む。

### Q6. 売上ゼロでモチベーションが続きません

**A**: 売上 0 円期間の楽しみ方:
- ユーザー数（無料も含む）で進捗測る
- 「自分のサービスを使った人がいる」だけで価値
- フォロワー数・記事 PV など、別の指標を見る
- 半年は売上ゼロでもおかしくない、と心の準備

## 修了の章: あなたへ

8 週間お疲れさまでした。

カリキュラムを完走したあなたは、もう **「AI 個人開発者」** です。

8 週間前のあなたと比べて、何が変わったでしょうか:

- ✅ Claude Code が使いこなせるようになった
- ✅ 自分のサービスを作って公開した
- ✅ 決済機能まで実装した
- ✅ AI 機能を組み込んだ
- ✅ 最初のユーザーに使ってもらった

**これが「ビジネスを 0 から 1 に立ち上げる経験」** です。
普通の会社員人生では、なかなか経験できないことです。

### あなたが手に入れた「3 つの財産」

#### 1. 「自分は作れる」という自信
これは何にも代えがたい。今後、何を見ても「これも作れそう」と思えます。

#### 2. 「Claude Code 操縦スキル」
8 週間で身につけたフレーズ集 150 個と、上手に頼むコツ。
これは **一生もの** のスキルです。

#### 3. 「最初のサービスを作った経験」
これがある人と、無い人では、人生の選択肢が **桁違い** に広がります。

### これからのあなたに伝えたいこと

> **「8 週間の挑戦が、人生最大のターニングポイントになる」**

これは大袈裟ではありません。
過去の受講者の中には、卒業半年後に **会社を辞めて独立した方**、**月収が 10 倍になった方**、**配偶者と一緒にサービス運営を始めた方** もいます。

今のあなたは、それらの人と **同じスタートライン** に立っています。

### 卒業生コミュニティで、また会いましょう

卒業生コミュニティは、**「8 週間で身につけたものを、続けて伸ばすための場」** です。

- 月 1 回のオンライン勉強会
