# Week 3: デザインを AI に注文する

> **このWeekのゴール**: 人に見せても恥ずかしくない、プロ品質のデザインに仕上げる
> **所要時間**: 動画 50 分 + 教材 90 分 + 宿題 30 分
> **このWeekで身につく力**: 「デザインの素人」が「プロっぽい見た目」を 5 分で生み出せる魔法

---

## このWeek を始める前に

### Week 2 までの状態

Week 2 で作ったあなたのサイトは、**「内容は揃っているけど見た目がまだ素人っぽい」** はずです。これは正常です。
Week 3 でこれを **「人に URL を送れる品質」** に引き上げます。

### Web デザインの真実

> 「デザインセンスが無いと無理」は、**もう過去の話**です。

なぜか:
- 「プロっぽさ」の正体は **「余白・配色・フォント」の 3 つで 80%** 決まる
- これらは Claude Code に **「プロっぽくして」** と頼むだけで自動で揃います
- センスではなく **「正しい頼み方」を知っているか** だけが差になります

Week 3 でその「頼み方」を覚えれば、あなたは一生 **「デザインで悩まない」** 人になります。

## レッスン 3-1: ビフォーアフターで学ぶ（15 分）

### 「もっとプロっぽくして」の威力

実際にやってみましょう。Week 2 で作ったサイトを開いて、Claude Code を起動。

```
このサイト全体を、もっとプロっぽい見た目にして。
余白・配色・フォントを整えて、高級感のある印象にして。
```

→ 1〜2 分で、別人のように整います。

### プロっぽさの 4 大要素

| 要素 | 素人っぽさ | プロっぽさ |
|------|-----------|-----------|
| **余白** | 詰まっている | ゆとりがある |
| **配色** | 多色使い・原色 | 2〜3 色に絞る・落ち着いた色 |
| **フォント** | デフォルト・MS ゴシック等 | Noto Sans / Inter / Helvetica 等 |
| **写真** | 個人撮影感 | プロ品質（フリー素材で OK） |

これらを Claude に「○○にして」と頼めるようになることが、Week 3 の本質です。

### 5 つのビフォーアフター実例

#### 例 ①: 個人ブログ
```
ビフォー: 詰まったレイアウト・黒文字オンリー・フォントはMSゴシック
↓「Notion 風のミニマルなデザインにして」
アフター: たっぷりの余白・グレー基調・読みやすいフォント・サムネイル統一
```

#### 例 ②: カフェのサイト
```
ビフォー: 写真がバラバラ・色が多い
↓「Apple のサイトみたいな高級感のあるデザインにして」
アフター: 大きな 1 枚写真・モノクロ基調・洗練された印象
```

#### 例 ③: SaaS の LP
```
ビフォー: 機能羅列・ボタン乱立
↓「Stripe や Linear のような、信頼感のあるシンプルなデザインにして」
アフター: 余白でリズム・ゴールド or 青のアクセント・大きな CTA
```

## レッスン 3-2: 色とフォントの頼み方（15 分）

### 色の頼み方

#### パターン ①: 雰囲気で頼む（一番楽）

```
「高級感のある黒×ゴールドの配色にして」
「ナチュラルでオーガニックな印象（ベージュ・木目・グリーン）にして」
「ポップで明るい配色（パステルカラー）にして」
「クールで知的な印象（ダークブルー×シルバー）にして」
「温かみのあるオレンジ系（柿色・キャメル）にして」
```

#### パターン ②: 具体的なカラーコードで頼む

```
「メインカラーを #c9a84c（ゴールド）に変えて」
「アクセントカラーを #ef4444（赤）に変えて」
「背景を #0a0a0f（ほぼ黒）にして」
```

#### パターン ③: 参考サイトを見せる

```
「https://stripe.com の配色を真似して」
「Apple のサイトみたいな白×グレー基調にして」
```

#### 配色のコツ

> **「3 色以内」** がプロの鉄則
- メインカラー（一番使う）
- アクセントカラー（CTA・強調）
- ベース色（背景・余白）

### フォントの頼み方

#### よく使う頼み方

```
「フォントを Noto Sans JP に変えて。見出しは太字で」
「セリフ体（明朝体）の高級感のあるフォントにして」
「丸ゴシックで親しみやすい印象にして」
「英語部分は Inter フォントを使って」
```

#### フォント選びの早見表

| 印象 | おすすめフォント |
|------|----------------|
| 信頼感・ビジネス | Noto Sans JP / Inter |
| 高級感・上品 | Noto Serif JP / Playfair Display |
| 親しみやすさ | M PLUS Rounded 1c |
| クリエイティブ | Zen Maru Gothic / Caveat |
| ミニマル | Helvetica / Roboto |

## レッスン 3-3: レイアウトを変える（15 分）

### よく使うレイアウト指示

```
「ヒーローセクションを全画面にして」
「2 カラムレイアウトにして（左 30% に画像、右 70% に文章）」
「カードを横並び 3 列にして」
「グリッドレイアウトで 6 つのカードを並べて」
「ジグザグレイアウト（左右交互）に並べて」
「画面中央に固定で表示して」
```

### セクション間の調整

```
「セクション間の余白をもっと広く取って」
「セクション背景を、奇数番目はグレー、偶数番目は白で交互にして」
「ヒーローと次のセクションの境界をグラデーションで繋いで」
```

### コンテナの整え方

```
「コンテンツの最大幅を 1280px に揃えて」
「左右の余白を 64px 取って」
「カード内の余白を 24px に統一して」
```

## レッスン 3-4: 参考サイトを見せる技（15 分）

### この技 1 つで、デザイン力が 10 倍になります

Claude Code は **「URL を見せる」** か **「スクリーンショットを見せる」** だけで、参考サイトを真似してくれます。

#### URL で頼む

```
このサイトみたいな雰囲気にして:
https://stripe.com
```

→ Claude が URL を見て、その雰囲気を再現します。

#### スクリーンショットで頼む

1. 参考サイトのスクリーンショットを撮る（Mac: `⌘+Shift+4`、Windows: `Win+Shift+S`）
2. ターミナルにドラッグ&ドロップ
3. 「このスクショみたいな感じにして」と頼む

→ Claude が画像を解析して、似たデザインを作ります。

- [Notion](https://notion.so) — 親しみ×洗練

#### EC・ブランドサイト

## レッスン 3-5: スマホ対応を頼む（10 分）

### スマホ対応（レスポンシブ）の頼み方

```
スマホで見ても綺麗に表示されるようにして。

具体的には:
- ヘッダーはハンバーガーメニュー化
- カードは縦並び 1 列に
- 文字サイズ・余白を適切に調整
- ボタンはタップしやすいサイズに
- 画像は画面幅に合わせて自動でリサイズ
```

### スマホ確認の方法

#### 方法 ①: ブラウザの開発者ツール（一番簡単）
- ブラウザでサイトを開く
- 右クリック → 「検証」
- 上部の「📱 スマホアイコン」をクリック
- iPhone/Android のサイズで確認できる

#### 方法 ②: 自分のスマホで確認（Week 6 以降）
- Vercel に公開後、URL をスマホで開く

### スマホで確認すべき 5 ポイント

```
- [ ] 文字が小さすぎない（14px 以上）
- [ ] ボタンがタップしやすい（44px 四方以上）
- [ ] 横スクロールが発生していない
- [ ] ハンバーガーメニューが動く
- [ ] 画像が画面からはみ出ていない
```

問題があれば:
```
スマホで見たら○○がはみ出ています。直して。
```

## レッスン 3-6: アニメーション・動きをつける（10 分）

ちょっとした動きで **「お、プロが作ったサイトだ」** という印象になります。

### 鉄板アニメーション 5 種

```
「スクロールしたら、要素がふわっとフェードインするようにして」
「ボタンにホバーした時、少し浮き上がるエフェクトをつけて」
「ヒーローのテキストが、ページ読み込み時に上から降りてくるアニメーションをつけて」
「画像にホバーすると、少しズームするようにして」
「ナビゲーションメニューを、スクロールで現れたり隠れたりするようにして」
```

### やりすぎ注意

> アニメーションは **「3 種類以内」** が鉄則。
> 全部に動きをつけると、逆に素人っぽくなります。

## レッスン 3-7: 実習: 自分のサイトを磨く（30 分）

### 進め方（推奨順序）

```
Step 1: 全体トーンを決める（高級感 / ナチュラル / ポップ 等）
       ↓
Step 2: 配色を整える（メイン・アクセント・ベース）
       ↓
Step 3: フォントを選ぶ
       ↓
Step 4: 余白とレイアウトを調整
       ↓
Step 5: ヒーロー・CTA・カードを「主役」として目立たせる
       ↓
Step 6: スマホ対応
       ↓
Step 7: アニメーション 1〜3 種類を追加
```

### 一気に頼むテンプレ

```
このサイト全体を、以下の方針でプロ品質に仕上げて。

【方向性】
- [高級感 / ナチュラル / ポップ / ミニマル / クリエイティブ から選ぶ]

【配色】
- メインカラー: [色]
- アクセントカラー: [色]
- ベース: [白系 / 黒系]

【フォント】
- 見出し: [Noto Serif / Inter 等]
- 本文: [Noto Sans / Helvetica 等]

【参考にしたいサイト】
- [URL を 1〜2 個]

【スマホ対応】
- ハンバーガーメニュー化
- 全要素レスポンシブ

【アニメーション】
- スクロールフェードイン
- ボタンホバーで浮き上がり
```

→ 1 回でかなり整います。

## 「プロっぽく見える」チェックリスト 15 項目

仕上げの最終チェック。**全部 ✅ になれば、人に URL を送れる品質** です。

### 余白（4 項目）
- [ ] セクション間に十分な余白（80〜120px）がある
- [ ] コンテンツの左右に余白（最低 16px、PC では 64px）がある
- [ ] カード・ボタン内の余白（パディング）が適切
- [ ] 行間（line-height）が 1.6〜1.8 に設定されている

### 配色（3 項目）
- [ ] 使用色が 3〜4 色以内に絞られている
- [ ] 文字色と背景色のコントラストが十分（読みやすい）
- [ ] CTA ボタンの色が一目で分かる（目立つ）

### フォント（3 項目）
- [ ] 日本語フォントが指定されている（デフォルトの MS ゴシック等になっていない）
- [ ] 見出しと本文でフォントサイズに階層がある
- [ ] フォントウェイト（太さ）が使い分けられている

### レイアウト（3 項目）
- [ ] スマホで横スクロールしない
- [ ] ボタンが指でタップしやすいサイズ（44px 四方以上）
- [ ] ナビゲーションがスマホでハンバーガーメニュー化

### 仕上げ（2 項目）
- [ ] スクロールアニメーションが入っている
- [ ] ボタンにホバーエフェクトがある

## トラブルシューティング

### Q1. 「プロっぽくして」と頼んだら、別物のサイトになりました

**A**: 内容を残したまま見た目だけ変えるよう指示し直してください:
```
内容（テキスト・構成）はそのまま、見た目だけプロっぽくして
```

### Q2. 自分の好きな色がうまく伝えられません

**A**: 色の名前 + 雰囲気で伝えると◎:
```
「ターコイズブルーで、爽やかな海みたいな雰囲気にして」
「くすみピンクで、大人っぽい雰囲気にして」
```

### Q3. 参考サイトの URL を見せても、思った通りにならない

**A**: より具体的に何を真似してほしいか伝える:
```
このサイトの「配色とヒーローセクションのレイアウト」だけを真似して。
内容は今のままで:
https://stripe.com
```

### Q4. アニメーションが多すぎて目が疲れます

**A**: 「シンプルに」と頼んでください:
```
アニメーションを減らして、もっと落ち着いた印象にして
```

### Q5. スマホで見ると、まだ崩れている部分があります

**A**: 具体的に伝える:
```
スマホで見ると、ヘッダーのロゴと検索ボタンが重なっています。直して。
```
スクショを添付するとより確実。

## コラム: 「自分のセンスじゃ無理」と思っているあなたへ

私は今でも **「センスが無い」** 自覚があります。

絵心ゼロ、ファッションも妻に選んでもらう、部屋のインテリアもよくわからない。
そんな人間が、なぜ AI 個人開発で「綺麗なサイト」を量産できるのか？

**答え: AI に任せれば、センスは要らない**から。

このカリキュラムで教えているのは、**「AI に上手に頼んで、プロっぽい結果を引き出す技術」**です。
これは「センス」ではなく「フレーズ」と「順序」です。
誰でも身につきます。

Week 3 が終わった今、あなたのサイトは恐らく **「人に URL を送れる品質」** になっているはず。
もし「まだ自信が無い」と思うなら、一度コミュニティに URL を投稿してみてください。仲間や講師がフィードバックします。
