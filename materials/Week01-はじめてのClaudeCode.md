## レッスン 1-1: Claude Code って何？（10 分）

### 2 分でわかる Claude Code

**Claude Code は、AI に日本語でお願いするだけで、PC の中で本物のアプリやサービスを作ってくれるツール**です。

```
あなた:    「シンプルなメモ帳アプリを作って」
Claude:   （5 秒考えて）「はい、作ります」
          （PC のフォルダに勝手にファイルを作る）
          「できました。実行してみてください」
あなた:    実行する → メモ帳アプリが立ち上がる ✨
```

これが Claude Code です。**コードを 1 行も書きません。日本語で頼むだけ**。

### 「ChatGPT との違い」を 30 秒で説明

| | ChatGPT / 普通の Claude | Claude Code |
|---|------------------------|-------------|
| できること | 文章を書いてくれる・質問に答える | **PC の中でファイルを作る・コードを書く・アプリを動かす** |
| 結果 | 画面に文字が出る | **本物のアプリ・サイトが PC の中に出来る** |
| 例えるなら | 知識豊富な相談相手 | **手を動かしてくれる優秀な部下** |

ChatGPT は「やり方を教えてくれる」だけ。Claude Code は **「実際にやってくれる」**。
これが革命的な違いです。

### なぜ今、これがチャンスなのか

**10 年前 vs 今**

```
10 年前:
  Web サービスを作る = プログラミング言語を 2 年学ぶ + 100 万円かけて外注
  → ハードルが高すぎて、ほとんどの人が諦めていた

今（Claude Code 登場後）:
  Web サービスを作る = 日本語で頼む
  → 8 週間で誰でも作れる
```

**今、まだ気づいていない人がほとんどです**。だから今やる人が「圧倒的に得をする時代」になりました。

このチャンスに気づいて、AI Builders Lab に来てくれた **あなたは、もう半分勝っています**。

## レッスン 1-2: PC にインストールしよう（20 分）

### Mac の方

#### ステップ ①: ターミナルを開く

1. キーボードで **`⌘ (Command) + Space`** を押す
2. 検索窓が出てくるので **「ターミナル」** と打つ
3. **Enter** を押す
4. 黒い画面（白い画面）が出てくる ← これがターミナルです

> 💡 **ターミナルの正体**: 「PC に文字でお願いごとをする画面」です。怖くないです。

#### ステップ ②: Node.js をインストール

ターミナルに以下をコピペして **Enter**:

```bash
brew install node
```

> ❓ **「brew コマンドが見つかりません」と出たら？**
> Homebrew が入っていません。先に以下を実行（コピペ）:
> ```bash
> /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
> ```
> 完了したら、ターミナルを一度閉じて、もう一度開いてから `brew install node` を実行。

#### ステップ ③: Claude Code をインストール

ターミナルに以下をコピペして **Enter**:

```bash
npm install -g @anthropic-ai/claude-code
```

数十秒〜1 分で完了します。

#### ステップ ④: 起動してみる

ターミナルに以下をコピペして **Enter**:

```bash
claude
```

→ 初回は Anthropic アカウントでのログインが求められます。ブラウザが開くので、そのまま画面の指示に従ってください。

完了すると、ターミナルに **`>` (プロンプト)** が表示されます。**ここに日本語を打ち込めるようになったら成功** です。

---

### Windows の方

#### ステップ ①: PowerShell を開く

1. キーボードの **Windows キー** を押す
2. **「PowerShell」** と打つ
3. **Enter** を押す

> 💡 黒い画面ではなく、青い画面が出てくる場合もあります。それでも大丈夫です。

#### ステップ ②: Node.js をインストール

ブラウザで [https://nodejs.org](https://nodejs.org) を開き、**「LTS」と書いてある方のインストーラ** をダウンロード。
ダウンロードしたファイル (`.msi`) をダブルクリックして、画面の通り **「次へ」「次へ」** で完了。

#### ステップ ③: Claude Code をインストール

PowerShell に戻って、以下をコピペして **Enter**:

```powershell
npm install -g @anthropic-ai/claude-code
```

#### ステップ ④: 起動してみる

```powershell
claude
```

ブラウザでログイン → 完了したら、`>` プロンプトが出ます。

---

### インストールできない時のチェックリスト

| 症状 | 対処法 |
|------|--------|
| `npm: command not found` | Node.js のインストールに失敗。再インストール。Mac は `brew install node`、Win は nodejs.org から |
| `permission denied` | Mac の場合、コマンド頭に `sudo` をつけて再実行（パスワード求められる） |
| `claude: command not found` | ターミナルを一度閉じて、もう一度開いて再試行 |
| ログイン画面が開かない | ターミナルに表示される URL を、ブラウザに手でコピペ |
| その他のエラー | エラー文を画像で撮って **コミュニティに投稿** → 講師か他の受講者が即答します |

## レッスン 1-5: 上手に頼む 5 つのコツ（15 分）

ここからは、Claude Code に **「上手にお願いするコツ」** を学びます。これが上達の一番の近道。

### コツ ①: 具体的に頼む

| ❌ 微妙な頼み方 | ✅ 良い頼み方 |
|----------------|---------------|
| 「メモ帳アプリ作って」 | 「白い背景・黒い文字で、左にタイトル一覧、右に本文編集エリアがあるメモ帳アプリを作って。タイトルをクリックすると右に本文が表示される」 |
| 「色変えて」 | 「ボタンの色を青系（#3b82f6）に変えて」 |
| 「もっと良くして」 | 「文字サイズを大きく（18px くらい）して、行間も広げて」 |

**ポイント**: 数字・色・場所・大きさを「具体的に」言うと、思い通りに作ってくれます。

### コツ ②: 一度に 1 つだけ頼む

```
❌ 「メモ帳作って、ログインも付けて、AI 検索もつけて、デザインを高級感あるやつにして、スマホ対応もして」

✅ Step 1: 「メモ帳アプリを作って」（出来てから...）
   Step 2: 「ログイン機能を追加して」（出来てから...）
   Step 3: 「AI 検索機能を追加して」
```

**理由**: AI も一度に多くを言われると、どれかを忘れたり、雑になったりします。**「1 つずつ・出来てから次」** が鉄則。

### コツ ③: 例を見せる

```
✅ 「Notion みたいな感じのメモアプリにして」
✅ 「Apple のサイトみたいな高級感のあるデザインにして」
✅ 「このサイトみたいなレイアウト: https://stripe.com 」
```

**人間が誰かに頼む時と同じ**。「○○みたいな感じ」と具体例を見せれば、AI も狙いが分かります。

### コツ ④: 修正は「ここを → こうして」

```
❌ 「なんか違う」

✅ 「タイトルの文字色を、今のグレーから黒に変えて」
✅ 「ボタンの位置を、画面の右下に固定して」
✅ 「ヘッダーの高さを 80px から 60px にして」
```

**「どこを」「どうしたいか」を明確に**。これだけで思い通りになります。

### コツ ⑤: 褒めてから直す

```
✅ 「全体的にすごく良いです！1 点だけ、サイドバーの幅をもう少し狭く（200px）してください」
```

実は AI 相手でも、**「良いところを認めてから直してもらう」**方が結果が良くなります。
理由は「直すべき場所だけが明確になり、他を壊さなくなるから」。
人間と同じです（笑）

## トラブルシューティング（よくあるつまずき）

### Q1. ターミナルで `claude` と打つと `command not found` が出ます

**A**: インストール直後はターミナルが古い情報のままなので、**ターミナルを一度閉じて、もう一度開いて** 再試行してください。それでもダメなら、もう一度 `npm install -g @anthropic-ai/claude-code` を実行。

### Q2. Claude Code がやたらと「許可しますか？」と聞いてきます

**A**: 正常です。ファイル作成や変更のたびに確認を求めます。
- **`y` Enter** = 許可
- **`n` Enter** = 拒否
- **`a` Enter** = このセッション中、全部許可（慣れたらこれで OK）

### Q3. ブラウザで HTML ファイルを開くと、文字化けします

**A**: ファイルの先頭に `<meta charset="UTF-8">` が無い可能性。Claude に「文字化けしてる。直して」と言えば直してくれます。

### Q4. アプリが動きません。何が悪いのか分かりません

**A**: 以下を Claude に伝えてください:
```
作ってもらったアプリが動きません。
画面に何も表示されない / エラーが出る（実際の症状）
スクリーンショット: [画像をドラッグ&ドロップ]
```
画面ごと見せれば、ほぼ 100% 直してくれます。

### Q5. インストールに 30 分以上かかります

**A**: ネットワークの問題です。Wi-Fi を再起動してリトライ。または、家族の協力を得て別の Wi-Fi で試す。

### Q6. お金がいくらかかるか不安です

**A**: Claude Pro の月額 **$20**（約 3,000 円）が固定です。これ以上はかかりません。Week 5 で API を使い始めると、追加で月 $5〜$10 程度。**合計でも 5,000〜6,000 円/月** に収まります。
