# AI Builders Lab — 教材へようこそ

## このアカデミーについて

ようこそ、AI Builders Lab へ。

このアカデミーは、**コードを 1 行も書かずに、自分の Web サービスを作って公開し、お金を受け取れる状態になる**までを、8 週間で身につけてもらう場所です。

教える側は、ノーコードで本格競馬予想 AI を作り、その後も AI 個人開発で 10 以上のプロダクトをリリースしてきた現役開発者。教える内容は、その全プロセスで使ってきた **「Claude Code に日本語で頼む」だけの開発手法** です。

---

## 受講者へのメッセージ

> プログラミング言語を覚える必要はありません。
> ターミナルの黒い画面が怖くなくなる必要もありません。
> 必要なのは、**「日本語で頼む勇気」と「最初の 1 週を踏み出すこと」だけ**です。

8 週間後、あなたの手元には:

- ✅ 自分のドメインで公開された Web サービス
- ✅ 決済機能が動く有料コンテンツ
- ✅ AI 機能が組み込まれた、他にない自分だけのプロダクト
- ✅ それを SNS で告知して最初のお客さんを獲得した経験

これが、当たり前のように手に入ります。

## 教材の使い方

### 1. 1 週ずつ、順番に進めてください

各 Week は前の Week の内容を前提にしています。飛ばさず、順番に。
もし「分からない」が 1 つでも出たら、その場で `Q&A` セクションを見るか、コミュニティで質問してください。**1 週間に 1 回もつまずかない人はいません**。つまずきは恥ではなく、学びの始まりです。

### 2. 動画 → 教材 → 実習 の 3 ステップ

```
① 動画を見る              （40〜50分／週）
       ↓
② この教材を読みながら、自分の PC で同じことをやる   （60〜90分／週）
       ↓
③ 宿題を出す（コミュニティに投稿）        （30分／週）
```

**合計: 週 2〜3 時間**。これだけです。

### 3. フレーズ集は手元に置いてください

各 Week には **「フレーズ集」**が付いています。「こう言えば → こうなる」の対応表で、これがあなたの **最大の武器** になります。

- 全 8 巻、計 **150 以上のフレーズ**
- 印刷して机の横に置く / スマホで開いておく / Notion にまとめ直す等、自分の使いやすい形で
- 卒業後もずっと使える「Claude Code 操縦マニュアル」になります

### 4. 詰まったら 3 つの順番で対処

```
① まず Claude Code に「エラーが出た」と画面ごとコピペして渡す
       ↓
② それでも解決しなければ、各 Week 末尾の「トラブルシューティング」を見る
       ↓
③ それでもダメなら、コミュニティに投稿
```

**1 人で抱え込まない**。これが最大のコツです。

## 受講前の心構え（大事）

### 1. 「動けば勝ち」だと思ってください

「この書き方は綺麗じゃないかも」「もっと上手な方法があるかも」—— そんなことは **一切気にしなくていい**です。

プロのエンジニアでも、「動くコード」は最高に価値があります。動いたら、それは正解です。あとから AI に「もっと綺麗にして」と頼めば、AI が綺麗にしてくれます。

### 2. 「自分には才能がないかも」は禁句

このカリキュラムは、**プログラミング経験ゼロの人**が **8 週間で本物の Web サービスを公開できる**ように設計されています。あなたが特別優秀である必要はありません。

過去の受講者の声:

> 「PC は普通に使えるけど、コードなんて書いたことない 40 代主婦です。3 週目で自分のサイトが画面に表示された時、本当に泣きました。」

> 「副業で月 5 万円欲しくて始めました。8 週目に告知したサービスが、初月で 3 万円の売上に。来月には 5 万円超えそうです。」

### 3. つまずきは「全員が通る道」

各 Week にはトラブルシューティングがあります。そこに書いてあるエラーは、**過去の受講者全員が一度はぶつかったもの**です。あなただけじゃない。

エラーが出たら、それは **「あ、自分も全員と同じ道を通ってる」というサイン** だと思ってください。

## 講師から、あなたへ

私は今でこそ「AI で何でも作れる人」と呼ばれていますが、**始めた時はあなたと同じ「プログラミング経験ゼロ」**でした。

最初に作ったのが、本格的な競馬予想 AI です。「競馬で勝ちたい」という不純な動機 (笑) から始まりましたが、結果としてそれが、私の人生を変えました。

「コードを書けない自分には無理」と思っていた壁は、**Claude Code の登場で完全に消えました**。日本語で頼むだけで、本物のサービスが作れる。これがどれほど革命的か、8 週間でわかってもらえると思います。

8 週間後、あなたが作った Web サービスを見せてもらえる日を、楽しみにしています。
