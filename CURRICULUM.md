# AI Builders Lab — 8週間カリキュラム詳細

**コンセプト**: プログラミング経験ゼロから、8週間でAIプロダクトを完成・リリースする
**前提**: 受講者はコードを1行も書かない。全てAI（Claude Code等）への指示で開発する
**最終成果物**: 受講者オリジナルのWebアプリを1つ以上リリースして公開する

---

## Week 1: AIツールの使い方 — 「話すだけで作れる」を体験する

### テーマ

|---|---------|------|------|
| 1-1 | イントロダクション | 15分 | AI個人開発とは何か。なぜ今がチャンスなのか。8週間のゴール共有 |

| 1-6 | 実習: プロフィールサイトを作る | 45分 | 自分のプロフィールサイトをClaude Codeで作成 → ローカルで表示確認 |

## 動画制作方針

### 制作手段

| 方法 | 内容 |
|------|------|
| スライド | Google Slides or Canva（ダーク+ゴールド統一デザイン） |
| 画面収録 | OBS Studio でデスクトップ録画 |
| 音声 | 仁さんのナレーション or AI音声（Qwen3-TTS / ElevenLabs） |
| 編集 | DaVinci Resolve（無料）or Remotion（自動化の場合） |
| サムネイル | Canva / FLUX2（AI画像生成） |

### AI音声で自動制作する場合（Phase 7以降で検討）

`~/claude-code-video-toolkit` のパイプラインを流用：
1. Claude API でスライドの台本を生成
2. FLUX2 でスライド画像を生成（Modal Cloud GPU）
3. Qwen3-TTS でナレーション音声を生成（Modal Cloud GPU）
4. Remotion で合成→レンダリング
5. Cloudflare Stream にアップロード

→ 全8週分（8本×約60分）の動画を自動生成可能（要品質チェック）

## Supabase テーブル設計（講座コンテンツ管理）

```sql
-- 講座コンテンツ
CREATE TABLE courses (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  week INT NOT NULL,
  lesson_number TEXT NOT NULL,       -- "1-1", "1-2" etc
  title TEXT NOT NULL,
  description TEXT,
  video_url TEXT,                     -- Cloudflare Stream URL
  duration_minutes INT DEFAULT 0,
  sort_order INT DEFAULT 0,
  is_published BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 配布資料
CREATE TABLE materials (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  week INT NOT NULL,
  title TEXT NOT NULL,
  description TEXT,
  file_url TEXT NOT NULL,             -- Cloudflare R2 or Supabase Storage
  file_type TEXT DEFAULT 'pdf',       -- pdf / md / zip / py
  sort_order INT DEFAULT 0,
  is_published BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);

-- 受講者の進捗
CREATE TABLE progress (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id),
  course_id UUID REFERENCES courses(id),
  completed BOOLEAN DEFAULT false,
  completed_at TIMESTAMPTZ,
  UNIQUE(user_id, course_id)
);

-- 宿題提出
CREATE TABLE submissions (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id),
  week INT NOT NULL,
  content TEXT,                       -- Markdown or URL
  screenshot_url TEXT,
  submitted_at TIMESTAMPTZ DEFAULT now(),
  feedback TEXT,                      -- 講師からのフィードバック
  feedback_at TIMESTAMPTZ
);
```
