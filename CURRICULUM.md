# AI Builders Lab — 8週間カリキュラム（初心者完全対応版）

**コンセプト**: Claude Code への日本語指示だけで、自分のWebサービスを作って公開・収益化する
**前提**: 受講者はコードを1行も書かない。全て「日本語でAIに頼む」だけ
**ターゲット**: プログラミング経験ゼロ。PCは普通に使える人
**最終成果物**: 自分のWebサービスがネット上に公開され、お金を受け取れる状態になっている

---

## 全体の流れ

```
Week 1    Week 2    Week 3    Week 4    Week 5    Week 6    Week 7    Week 8
準備      体験      見た目    機能追加  AI搭載    公開      収益化    集客
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
入れる    作る      磨く      動かす    賢くする  見せる    稼ぐ      広める
```

## 動画の作り方

各Weekの動画は以下の構成：

| パート | 時間 | 内容 |
|--------|------|------|
| 今週のゴール | 2分 | 「今週が終わると○○ができるようになります」 |
| 実演（画面共有） | 30〜40分 | Claude Codeに話しかけながら実際に作る過程をそのまま見せる |
| フレーズ集の使い方 | 5分 | 今週のフレーズ集から重要なものをピックアップ |
| 宿題の説明 | 3分 | 今週の宿題・提出方法・締切 |
| **合計** | **約40〜50分/Week** | |

**制作のポイント**:
- スライドは最小限。**画面共有でClaude Codeと会話する映像がメイン**
- 「こう言ったらこうなった」のライブ感が最大の売り
- 編集は軽めでOK。「リアル感」が価値
